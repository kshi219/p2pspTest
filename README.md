# A console version of the P2PSP entities (Splitter, Peer, etc) #

## Cloning
To clone this projects with all submodules, make shure you clone with `--recursive` argument:
`git clone --recursive ...`


## Pre-requisites
### Linux 
```
$ [sudo] apt-get install libboost-all-dev
```
## Build from Source

```
./make.py
```
